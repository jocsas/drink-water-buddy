Put the Darth Vader theme images in this folder using these exact names:

- `idle.png`: character waiting/standing.
- `drinking.png`: character drinking or celebrating hydration.
- `tray.png`: optional menu bar icon for this theme.

If `drinking.png` is missing, the app uses `idle.png` as a temporary fallback.
If `tray.png` is missing, the app uses the default theme tray icon.
